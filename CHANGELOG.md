# Changelog

## 1.0.0 (2026-02-21)

### Features

* Animated loot feed with quality-colored borders and glow
* XP gain toast notifications with stacking
* ElvUI skin matching (fonts, textures, borders)
* LibSharedMedia support for custom fonts, textures, and sounds
* Configurable filters (quality, self/group loot, currency, gold, quest items, XP)
* Combat deferral with queue flush
* Minimap button via LibDBIcon
* Draggable anchor positioning with lock/unlock
* Duplicate item stacking
* Full AceConfig GUI with slash commands (/dt)
* Hover to pause fade timer and show tooltip
* Shift-click to link items in chat
* Configurable entrance animations (direction, distance, speed)
* Optional sound notifications via LibSharedMedia
